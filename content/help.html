Help

If you are having a problem using Prometheus, there are several ways to get help from the community.

Documentation

The official documentation for Prometheus is extensive, and covers the core system in depth.

IRC

IRC is good for getting interactive help. As our users are spread around the world, please be patient and hang around for a while if you do not get an immediate answer.

We are on #prometheus on irc.freenode.net. If you do not have an IRC client, you can use Webchat from your browser.

Mailing List

The prometheus-developers Google Group is good for longer or more complicated questions.
