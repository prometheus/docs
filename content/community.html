Community

Prometheus is developed in the open and has a growing community outside of SoundCloud. Here are some of the channels we use to communicate and contribute:

Mailing list: prometheus-developers Google Group

Twitter: PrometheusIO

IRC: #prometheus on irc.freenode.net

Issue tracker: We use the GitHub issue tracker for the various Prometheus repositories

Contributing

We welcome community contributions! Please see the CONTRIBUTING.md file in the respective Prometheus repository for instructions on how to submit changes. If you are planning on making more elaborate or controversial changes, please discuss them on the mailing list before sending a pull request.
